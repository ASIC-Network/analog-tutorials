### Toolchain
Getting started with open-source tools. For analog, the basic flow is xschem for simulation and spice netlist generation to klayout for, as stated, layout. Set up the container with PDK pre-installed here: [IIC-OSIC-TOOLS](https://github.com/iic-jku/iic-osic-tools)

### Order of tutorials
1. Inverter
2. Current mirror
3. Differential pair

> [!NOTE]
> This is a great [extensive course](https://iic-jku.github.io/analog-circuit-design/aicd.html#sec-gmid-method) on how to do analog design! These tutorials just serve to teach students how to run through schematic and layout on the open source tools (start to end on the workflow).
